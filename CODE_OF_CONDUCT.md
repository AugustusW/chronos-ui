# Code of Conduct

This project adopts the [Contributor Covenant](https://www.contributor-covenant.org/version/2/1/code_of_conduct/),
version 2.1.

## Reporting

There is no public maintainer email. To report unacceptable behavior confidentially, use the same
private channel documented in [SECURITY.md](SECURITY.md): this repository's **Security** tab →
**Report a vulnerability** (a private GitHub security advisory,
https://github.com/AugustusW/chronos-ui/security/advisories/new). It's the only confidential,
GitHub-native contact channel available for this project, and reports of any kind — not just
security ones — reach the maintainer privately that way. If confidentiality isn't a concern, a
regular GitHub issue is fine too.
